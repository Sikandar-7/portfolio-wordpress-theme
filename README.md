# Sikandar Abbas — Portfolio Website Theme

A custom WordPress theme built from scratch for my personal developer portfolio.

## ✨ Features

- **Custom WordPress Theme** — pure PHP/HTML/CSS/JS, no Elementor or page builders
- **Dark Mode Design** — premium dark aesthetic with purple accents
- **Custom Post Type** — Projects CPT with meta fields
- **Dynamic Projects Section** — fetches projects from WP admin
- **Smooth Animations** — scroll-triggered effects
- **Fully Responsive** — mobile and desktop
- **Skills Section** — with progress indicators
- **Contact Section** — with social links
